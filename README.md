# Stock Market Analysis of 4 Companies

This project performs data analysis on stock data from four different companies using Python. The goal is to explore trends, compare performance, and extract meaningful insights using data analysis and visualization techniques.

---

# Project Description

In this project, historical stock data of four companies is collected and analyzed. The analysis focuses on understanding:

- Stock price trends over time  
- Daily returns and volatility  
- Relationships between different stocks  
- Comparative performance  

---

# Dataset

The dataset contains stock market data such as:

- Date  
- Open price  
- Close price  
- High & Low prices  
- Volume  

---

# Tools & Technologies

- Python 
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Jupyter Notebook  

---

# Analysis Performed

## 1. Data Cleaning
- Handling missing values  
- Formatting date columns  
- Ensuring correct data types  

## 2. Exploratory Data Analysis (EDA)
- Summary statistics  
- Distribution of stock prices  
- Trend visualization  

## 3. Time Series Analysis
- Stock closing price over time  
- Moving averages  

## 4. Daily Returns
- Calculating percentage change  
- Comparing volatility between companies  

## 5. Correlation Analysis
- Heatmap to identify relationships between stocks  
- Understanding how stocks move together  

---

# Key Insights

- Identified trends in stock performance over time  
- Compared volatility across companies  
- Found correlations between stock movements  
- Visualized patterns for better decision-making
